# Projeto Spring Boot

Projeto de exemplo com Spring Boot para operações CRUD.

## 🛠️ Tecnologias

- Spring Boot
- Apache Tomcat
- PostgreSQL
- Maven
- H2 (testes)
- Heroku (deploy)
- Postman (testes de API)

## 📝 Endpoints

- **GET** `/produtos` - Lista todos.
- **GET** `/produtos/{id}` - Busca por ID.
- **POST** `/produtos` - Cria novo.
- **PUT** `/produtos/{id}` - Atualiza.
- **DELETE** `/produtos/{id}` - Remove.
